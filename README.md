# arilaverty.dev
Personal site the demonstrates VueJS and integration with GCP services:
* Firestore
* Firebase Auth

## Features
* Signup
* Login
* Password Reset
* Image Upload
* Post Commenting
* Post Sharing / Liking

## Project setup
```
yarn install
```

### Compiles and hot-reloads for development
```
yarn serve
```

### Compiles and minifies for production
```
yarn build
```

### Lints and fixes files
```
yarn lint
```
